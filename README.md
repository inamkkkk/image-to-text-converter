# Image to Text Converter

A Flutter app that uses Google ML Kit to extract text from images.

## Features

*   Select image from gallery.
*   Take image using camera.
*   Extract text from image.
*   Display extracted text.

## Dependencies

*   `flutter`
*   `cupertino_icons`
*   `image_picker`
*   `google_ml_kit`
*   `provider`
*   `path_provider`

## Getting Started

1.  Clone the repository.
2.  Run `flutter pub get` to install dependencies.
3.  Run `flutter run` to start the app.

## Note

Remember to enable camera and photo library permissions in your AndroidManifest.xml and Info.plist files.
